# User Flow - SHOPEZ

## Overview
The user flow describes the step-by-step process a user follows while interacting with the ShopEZ e-commerce platform.

---

## User Flow Steps

1. **Registration**
   - Users start by creating an account on the platform.

2. **Login**
   - After registration, users log in using their credentials.

3. **Browse Products**
   - Once logged in, users can explore available products.

4. **Add to Cart**
   - Users can select products and add them to their cart.

5. **Place Order**
   - Users proceed to checkout by entering shipping address and payment details.

6. **Order Confirmation**
   - After placing the order, confirmation is provided.

7. **View Orders**
   - Users can view their order details in the profile section.

---

## Conclusion
This flow ensures a smooth and user-friendly shopping experience from registration to order tracking.
