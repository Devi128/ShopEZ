# Roles and Responsibilities - SHOPEZ

## Overview
This section defines the roles involved in the ShopEZ e-commerce system and their responsibilities.

---

## Roles

### 1. User
Represents customers who use the platform.

**Responsibilities:**
- Register and login to the platform
- Browse products
- Add products to cart
- Place orders
- View order details

---

### 2. Admin
Represents the administrator of the platform.

**Responsibilities:**
- Manage banner images
- Manage product categories
- Monitor platform activities

---

### 3. Products
Represents all available products in the system.

**Responsibilities:**
- Store product details such as name, price, and description
- Display products to users
- Maintain product availability

---

### 4. Cart
Represents the temporary storage of selected items.

**Responsibilities:**
- Store products added by users
- Manage product quantity
- Differentiate items using userId

---

### 5. Orders
Represents completed purchases.

**Responsibilities:**
- Store order details
- Track user purchases
- Maintain order history
